# isObservable

<h2 class="subtitle">Comprueba si el objeto es un Observable de RxJS</h2>

### Firma

`isObservable<T>(obj: any): obj is Observable<T>`

### Parámetros

<table>
<tr><td>obj</td><td>El objeto que se desea comprobar.</td></tr>
</table>

### Retorna

`obj is Observable<T>`
